# Math-API
A rest math api project using spring boot java.

You can also find a test class.

URLs (replace "-" with the your number or your variable): 

1) http://localhost:8080/api/v1/math/add?a=-&b=-
2) http://localhost:8080/api/v1/math/subtract?a=-&b=-
3) http://localhost:8080/api/v1/math/multiply?a=-&b=-
4) http://localhost:8080/api/v1/math/divide?a=-&b=-

If you denided by 0, it should return an error with the message "Cannot divide by zero".
